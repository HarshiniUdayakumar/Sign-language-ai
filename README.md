# Offline Bidirectional Sign Language Communication System

An AI-powered offline system that enables two-way communication between
deaf and hearing users using sign language and speech.

## Features
- Sign language recognition using hand landmarks
- Offline speech-to-text
- Emotion-aware translation (planned)
- 3D signing avatar (planned)
- Low-resource optimized design

## Tech Stack
- Python
- OpenCV, MediaPipe
- TensorFlow Lite
- FastAPI
- React
